# UMS AI-Native Platform Deck

A cinematic, motion-driven HTML presentation on the **Unified Metering Service** — the platform powering Adobe's AI economy, and how we're building it AI-native.

**🔗 Live deck:** https://prahlad91.github.io/Decks/ums-ai-native-deck/

---

## About

A 10-slide editorial deck arguing that UMS isn't just *built for* AI — it's *built with* AI. Walks through the platform primitives (Authorize · Meter · Enforce), the scale we operate at, the FY26 launches we've enabled, and the velocity transformation across three fronts: client enablement, platform development, and internal operations.

## Tech

Single self-contained HTML file. CSS-only motion (GPU-accelerated transforms + opacity). Fonts from Google Fonts CDN: Fraunces (display serif), Inter Tight (UI), JetBrains Mono (labels). No build step, no dependencies, no analytics.
